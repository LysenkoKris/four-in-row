# Четыре в ряд

### Реализован стандартный режим игры «четыре в ряд». Особенности:
- Можно настраивать размер игрового поля и количество фишек для победы.
- Результаты и статистика побед сохраняются с помощью localStorage.
- Ведётся подсчёт количества побед у каждого игрока.
- В utils есть функция validator, вывод примера можно посмотреть в консоли во время игры.

### Как менеджер пакетов был выбран pnpm, сборщик Vite

Для установки зависимостей использовать

```pnpm install```

Для запуска проекта 

```pnpm dev```

Адрес для просмотра 

http://localhost:5173
